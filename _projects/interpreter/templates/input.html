IDLE LITE

Test your skills anywhere with this Interactive Python3 IDLE on the WEB!
Get ready to be armed with Py3 Gadgets going to be equipped in the next update.

Direction:

  • Any Error in your code would result in a blank output section.
  • Don"t type any GUI based program, it is only for development (almost like your IDLE) not for productive purposes.
  • modules not supported:
    • functools
    • setuptools
    • tkinter
    • web-frameworks
    • etc.
  • Supported modules:
    • Every module that returns printable output. (except that of turtle and that of pillow)
  • Terminal based processes would be done in server-side terminal. (in the web server)
  • In case of a ImportError or any module not found, write the following code and the error would be handled:
    from subprocess import call
    call("pip install module_name")

    wait for the output (which would be blank) and then try again.
